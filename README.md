# Nimietech

A simple app built with **Next.js**, **TypeScript**, and **Tailwind CSS**.

## 🚀 Features

- View Posts  
- Add Posts via Modal  
- View Users  
- Add Users via Modal
- Clean UI with Tailwind CSS

## 🛠 Tech Stack

- Next.js  
- TypeScript  
- Tailwind CSS  

## ▶️ Run Locally

```bash
npm install
npm run dev
